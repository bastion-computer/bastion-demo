# Prompt: Add Search Filtering

Add a text search box that filters issues by title, description, assignee, or ID.

Feature requirements:

- Add a search input near the status filter.
- Search should be case-insensitive.
- Status filtering and search should work together.
- Keep all filtering client-side for this demo.
- Add a small testable helper if it keeps the UI logic easy to understand.

Constraints:

- Do not add dependencies.
- Keep the mobile layout usable.
- Run `bun test` before you finish.
